# yuque-migration

语雀迁移工具，支持从一个语雀组织迁移到另一个语雀组织

导出逻辑基于 [yuque-dl](https://github.com/gxr404/yuque-dl) 修改
导入基于 语雀 api 实现

### Install

```bash
$ pnpm install
```

### Development

```bash
$ pnpm dev
```

### Build

```bash
# For macOS
$ pnpm build:mac
```
